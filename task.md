Тестовое задание на вакансию "Программист C#"

Необходимо написать консольное приложение, запускаемое командой dotnet Test.dll (т.е. использующее .NET Core любой версии) по следующим требованиям:
0. Приложение принимает параметры и вычисляет функцию от их значений, результат выводит в консоль. 
1. Функция выбирается из списка поддерживаемых функций, указанного в файле appsettings.json. Также в appsettings.json указана текущая выбранная функция.
2. Необходимо поддерживать 2 параметра командной строки с типом "double" и с именами "parameter_a" и "parameter_b"
3. Необходимо предусмотреть запуск приложения без указания этих параметров - тогда будут использоваться значения параметров, указанные в appsettings.json
4. Необходимо создать отдельный класс с методом TestFunction(double a, double b), результат вывода которого будет отображаться в консоли
5. В конфигурационном файле appsettings.json предусмотреть следующие варианты выбора функции для вычисления результата в TestFunction:
a) текущая выбранная функция 'add' приводит к вычислению суммы значений параметров a и b
b) текущая выбранная функция 'sub' приводит к вычислению разности значений параметров a и b
c) текущая выбранная функция 'mult' приводит к вычислению произведения значений параметров a и b

Результат в виде исходного кода и исполняемого кода проекта передается сотруднику кадрового агентства для проверки специалистами компании.
